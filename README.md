# TeamElogisolGrievanceAPI

This is a Node.js API project using Express.

## Project Structure

- controllers/
- models/
- routes/
- app.js
- package.json

## Getting Started

1. Install dependencies:
   ```powershell
   npm install
   ```
2. Start the server:
   ```powershell
   npm start
   ```

## API Example

- GET /api/health
  - Returns: `{ status: 'ok' }`
